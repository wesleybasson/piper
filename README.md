# PipeR

Lightweight pipeline utilities for .NET (core libraries and ASP.NET Core integrations).

Quick links
- License: `LICENSE`
- Contributing: `CONTRIBUTING.md`

Getting started

Prerequisites
- .NET SDK 7.0+ (or the SDK version used by the solution)

Build
```pwsh
dotnet build PipeR.sln
```

Run tests
```pwsh
dotnet test
```

Where to look
- Core library: `PipeR.Core/`
- ASP.NET Core integration: `PipeR.AspNetCore/`
